# Analysis-of-BigMart-Sales-Performance-Preprocessing

# 📌 Project Overview
This project analyzes BigMart sales data to identify how store types, outlet sizes, and product characteristics influence sales performance. The analysis includes data cleaning, preprocessing, statistical testing, and feature engineering to generate business insights.

# ❗ Problem Statement
BigMart operates multiple store formats with different business models. Understanding which outlet types generate the highest sales is essential for optimizing expansion and improving business performance.

# ❓ Business Questions
* Does store type significantly affect sales?
* Which outlet type generates the highest sales?
* Which outlet size performs best?
* What factors contribute to high-performing outlets?

# 📊 Dataset
* Source: Kaggle
* Records: 8,523 rows
* Features: 12 columns

Key features include Outlet Type, Outlet Size, Item Visibility, Item Weight, Item MRP, and Item Outlet Sales.

# 📈 Key Insights
* Store type significantly impacts sales performance.
* Supermarket Type 3 generates the highest average sales.
* Supermarket Type 1 is the most common outlet format.
* Small and Medium outlets dominate the BigMart network.
* Sales distribution is highly skewed, with a small number of outlets contributing most revenue.

# 💡 Recommendations
* Prioritize expansion of high-performing outlet types.
* Focus on Medium-sized outlets for stable sales performance.
* Use sales data to guide future store expansion strategies.
